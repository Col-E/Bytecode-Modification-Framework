Welcome to Bytecode Modification Framework (BMF).

The Bytecode Modification Framework is a project maintained by Matthew Coley and Matthew Dupraz and is designed to be a light-weight Java bytecode modification library designed to accurately portray and edit class files as they are without hiding any of the inner workings.

The TO-DO list

  • Method opcode editing (Requires finishing reading from CODE attribute)
  • Refactoring for class/field/method names and descriptions
  • Stack analysis capabilities (Requires finishing reading and writing from STACKMAPTABLE attribute)
  • Code cleanup and documentation
